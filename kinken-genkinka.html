金券の現金化とは？合法性を解説

金券の現金化は合法です

金券の現金化とは、手元にある商品券・ギフトカード・QUOカード・収入印紙などの金券を売却して現金に換えることです。「金券を売ってお金にするのは違法では？」と心配する方もいますが、自分が所有する金券を売ることは完全に合法です。

金券ショップは全国に数千店舗あり、古物営業法に基づく古物商許可を取得して合法的に営業しています。また、フリマアプリやオークションで個人が金券を出品・売却することも法律上問題ありません。

ただし、以下のケースには注意が必要です：

  • クレジットカードのショッピング枠で金券を購入して換金する行為 — カード会社の利用規約違反となり、カードの強制解約や一括返済を求められるリスクがあります
  • 転売目的で大量に仕入れて反復継続的に販売する行為 — 古物営業法の許可が必要になる場合があります
  • 盗品の売却 — 当然ながら犯罪行為です。拾得物の金券を売ることも遺失物横領罪に問われる可能性があります

通常の個人利用（贈り物でもらった商品券を換金する、余った収入印紙を売るなど）であれば、何の問題もなく現金化できます。先払い買取は違法？のページもあわせて参考にしてください。

現金化できる金券の種類

売却可能な主な金券一覧

現金化できる金券は多岐にわたります。以下が代表的な金券の種類です。種類によって換金率が大きく異なるため、お手持ちの金券がどの程度の価格で売れるか確認しておきましょう。

  • 商品券（JCBギフトカード・VJAギフトカード・UCギフトカード等） — 最もメジャーな金券。換金率は高めで95〜98%が相場
  • 百貨店商品券（三越・伊勢丹・高島屋・大丸等） — 信用力が高く換金率も優秀。95〜97%程度
  • QUOカード — コンビニで使える汎用性の高いカード。換金率は80〜90%程度
  • Amazonギフト券 — ネット上での需要が高い。換金率は80〜92%程度
  • 収入印紙 — ビジネス用途で需要が安定。換金率は90〜97%と高め
  • 切手（普通切手・記念切手） — 普通切手は80〜90%、記念切手はプレミアが付く場合も
  • ビール券・お米券 — スーパーで使える身近な金券。換金率は75〜85%程度
  • 図書カード — 書店限定のため需要はやや限定的。75〜85%程度
  • 旅行券（JTB・近畿日本ツーリスト等） — 高額面のものが多く換金率は90〜95%程度
  • テレホンカード — 需要が減少傾向で30〜60%と低め。プレミアカードは例外
  • 株主優待券 — 企業や時期により換金率は大きく変動。50〜90%

💡 換金率が高い金券の特徴

一般的に利用できる場所が多い金券ほど換金率が高くなります。JCBギフトカードやVJAギフトカードなどの全国共通商品券は使えるお店が多いため、常に高い換金率を維持しています。逆に、利用先が限られるテレホンカードや特定店舗のみで使える商品券は換金率が低くなる傾向があります。

方法①：金券ショップ

1

金券ショップで売るメリット・デメリット

金券ショップは駅前や繁華街に多く、店頭に金券を持ち込むだけでその場で現金を受け取れる最もシンプルな方法です。大黒屋・アクセスチケット・チケットキャビンなど全国チェーンの店舗が多数あります。

メリット：

  • ✓ その場で即現金化できる（手数料なし）
  • ✓ 換金率が比較的高い（JCBギフトカードで95〜98%）
  • ✓ 対面取引なので安心感がある
  • ✓ 面倒な出品作業や発送手続きが不要

デメリット：

  • ✗ 店舗まで行く必要がある（地方は店舗が少ない）
  • ✗ 営業時間内に行く必要がある（多くは10時〜19時）
  • ✗ 店舗によって換金率にばらつきがある
  • ✗ 少額や需要の低い金券は買取を断られる場合がある
💡 ポイント：金券ショップの換金率は店舗や地域で異なります。可能であれば事前に電話やWebサイトで換金率を確認してから訪問しましょう。駅前の激戦区ほど換金率が高い傾向があります。

方法②：先払い買取サービス

2

先払い買取で金券を売るメリット・デメリット

メリット：

  • ✓ 来店不要。スマホ1つで24時間申込み可能
  • ✓ 最短5分で銀行口座に入金
  • ✓ 商品を発送する前にお金を受け取れる
  • ✓ 地方在住で近くに金券ショップがない方にも便利

デメリット：

  • ✗ 金券ショップと比べると換金率がやや低い場合がある
  • ✗ 後日、商品を発送する必要がある
  • ✗ 業者選びを間違えるとトラブルになるリスクがある

先払い買取が初めての方は、手順や流れを事前に確認しておくと安心です。やり方ガイドで申込みから入金までの4ステップを詳しく解説しています。

方法③：ネット買取業者

3

ネット買取業者で金券を売るメリット・デメリット

ネット買取とは、Webサイトから申込みをして金券を郵送し、査定後に代金が振り込まれるサービスです。店舗を持たないため、金券ショップよりも高い換金率を提示できる業者もあります。

メリット：

  • ✓ 自宅から郵送するだけで完結する
  • ✓ 店舗型より換金率が高い場合がある
  • ✓ 大量の金券もまとめて売れる

デメリット：

  • ✗ 商品を先に郵送する必要があるため、入金まで1〜3営業日かかる
  • ✗ 送料がかかる場合がある（送料無料の業者もあり）
  • ✗ 郵送中の紛失リスクがある（書留推奨）
💡 ポイント：ネット買取は送料無料・振込手数料無料の業者を選ぶのがコツ。事前にWebサイトで見積もり金額を確認できるサービスが便利です。急ぎの場合は先払い買取のほうが早く現金を受け取れます。

方法④：フリマアプリ（メルカリ等）

4

フリマアプリで金券を売るメリット・デメリット

メルカリやラクマなどのフリマアプリで金券を出品して個人間で売買する方法です。自分で価格を設定できるため、高値で売れる可能性があります。

メリット：

  • ✓ 自分で価格を設定できるため高値を狙える
  • ✓ 利用者が多いため買い手が見つかりやすい
  • ✓ スマホだけで出品・取引が完結する

デメリット：

  • ✗ 販売手数料が10%かかる（メルカリの場合）
  • ✗ 売れるまでに時間がかかる場合がある
  • ✗ 出品作業・梱包・発送の手間がかかる
  • ✗ 購入者とのトラブル（値下げ交渉・クレーム等）が発生する可能性
  • ✗ 売上金の振込に手数料がかかる場合がある
💡 ポイント：フリマアプリでは額面の90〜95%程度で出品するのが売れやすい価格帯です。ただし10%の手数料＋送料を差し引くと、実質的な換金率は金券ショップと同程度か低くなる場合も。時間に余裕がある方向けの方法です。

方法⑤：オークション（ヤフオク等）

5

オークションで金券を売るメリット・デメリット

ヤフオクなどのオークションサイトで金券を出品し、入札者の競争で高値を狙う方法です。人気の金券であれば、フリマアプリ以上の高値が付くこともあります。

メリット：

  • ✓ 競争入札で思わぬ高値が付くことがある
  • ✓ 大量出品やまとめ売りがしやすい
  • ✓ 株主優待券など特殊な金券でも買い手が見つかりやすい

デメリット：

  • ✗ 出品手数料がかかる（ヤフオクは落札価格の8.8〜10%）
  • ✗ オークション終了まで数日かかる
  • ✗ 入札がなければ売れない可能性がある
  • ✗ 出品者の評価が低いと敬遠されがち
💡 ポイント：オークションは1円スタートで出品すると注目を集めやすいですが、入札が伸びなければ低価格で落札されるリスクもあります。最低落札価格を設定するか、相場を調べてから開始価格を決めましょう。

現金化する際の注意点

⚠️ 金券を現金化する前に知っておくべき5つの注意点

  • クレジットカード現金化は絶対NG — クレジットカードのショッピング枠で金券を購入し、すぐに売却して現金を得る行為はカード会社の規約違反です。発覚するとカードの強制解約・一括返済を求められます。最悪の場合、詐欺罪に問われる可能性もあります
  • 古物営業法を理解しておく — 個人が不要な金券を売る行為は合法ですが、営利目的で反復継続して売買する場合は古物商許可が必要になる場合があります。せどり・転売ビジネスとして金券を扱う場合は注意してください
  • 確定申告が必要になる場合がある — 個人の金券売却は通常非課税ですが、年間の譲渡益が50万円を超える場合や、継続的に売買して利益を得ている場合は確定申告が必要です。不安な方は税理士や税務署に相談しましょう
  • 有効期限のある金券は要確認 — ビール券やお米券には有効期限があるものがあります。期限切れの金券は換金できないため、手元に金券がある場合は早めに売却を検討しましょう
  • 偽造金券・盗難品の売買は犯罪 — 偽造された金券を売買する行為や、盗品と知りながら売却する行為は犯罪です。出所が不明な金券の取引は避けてください

💡 金券を高く売るための3つのコツ

少しでも高い換金率で金券を売るためのポイントをまとめました。

  • 複数の業者・方法で見積もりを取る — 金券ショップA店で95%、B店で97%ということは珍しくありません。手間を惜しまず複数社に査定を依頼しましょう
  • 需要が高まるタイミングを狙う — お歳暮・お中元シーズンの前やGW前は金券の需要が高まり、換金率がアップする傾向があります
  • 金券の状態を良好に保つ — 折れ・汚れ・シワがあると査定額が下がります。金券は購入時の袋やケースに入れて保管し、きれいな状態で売却しましょう

よくある質問

Q金券を現金化するのは違法ですか？ ▼
金券を現金化すること自体は違法ではありません。自分で購入した金券や贈り物として受け取った金券を売却することは、個人の財産処分として合法です。ただし、転売目的で大量に仕入れて反復継続的に販売する場合は古物営業法の許可が必要になる場合があります。また、クレジットカードのショッピング枠で金券を購入して換金する行為はカード会社の利用規約違反となるため避けてください。
Q金券の換金率が一番高いのはどの方法ですか？ ▼
一般的に換金率が最も高いのはフリマアプリやオークションで、額面の90〜98%程度で売れることがあります。ただし、売れるまでに時間がかかる点や手数料・送料がかかる点を考慮する必要があります。即日現金化したい場合は、金券ショップ（85〜97%程度）や先払い買取サービスがおすすめです。
Q使いかけのQUOカードや図書カードも換金できますか？ ▼
金券ショップでは使いかけ（残高が中途半端な）QUOカードや図書カードは買取不可となるケースが多いです。フリマアプリでは残高を明記すれば売れる場合もありますが、換金率は大幅に下がります。基本的には未使用・残高満額の状態で売却するのが最も有利です。
Q金券を売ったお金に税金はかかりますか？ ▼
個人が不要な金券を売却して得た利益は、譲渡所得として扱われます。生活用動産の譲渡は非課税ですが、年間50万円を超える譲渡益がある場合や、営利目的で継続的に売買している場合は確定申告が必要になることがあります。通常の個人利用の範囲であれば課税されるケースはほとんどありません。
Q金券ショップに持ち込む際、身分証明書は必要ですか？ ▼
古物営業法に基づき、買取金額が一定額以上の場合は本人確認が必要です。具体的な基準は店舗によって異なりますが、1万円以上の取引で身分証の提示を求められることが一般的です。運転免許証やマイナンバーカードなどの顔写真付き身分証を持参しておくとスムーズです。
Q先払い買取で金券を売るメリットは何ですか？ ▼
先払い買取の最大のメリットは、商品を発送する前に現金を受け取れることです。LINEで写真を送るだけで査定が完了し、最短5分で銀行口座に振り込まれます。来店不要でスマホ1つで完結するため、近くに金券ショップがない方や、すぐに現金が必要な方に特におすすめです。

金券を現金化する方法をおさらいしましょう：
